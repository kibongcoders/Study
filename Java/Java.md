자바의 주요 특징
자바의 주요 특징은 OOP(Object-Oriented Programming) 객체 지향 프로그래밍입니다.
반복되는 부분을 최소화해 단위별로 개발 가능하고 더 깨끗하고 유지보수하기 쉽고 안전하면서도 재사용이 가능한 코드를 만들 수 있게 됐습니다.

그러나 지나친 OOP를 강조하면 코드가 지나치게 장황하고 문법 또한 복잡해집니다.
